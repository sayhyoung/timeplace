# 인증샷 카메라 - 시간 장소 스탬프

## 앱 이름
인증샷 카메라 - 시간 장소 스탬프

## 짧은 설명
시간, 장소, 메모를 예쁘게 남기는 인증샷 스탬프 카메라

## 한 줄 소개 후보
- 찍는 순간 시간과 장소가 남는 인증샷 카메라
- 운동, 식단, 공부 기록을 예쁘게 남기는 스탬프 카메라
- 시간, 장소, 메모까지 한 장에 담는 인증샷 기록 앱

## 자세한 설명
인증샷 카메라는 사진에 시간, 장소, 메모 스탬프를 예쁘게 남길 수 있는 카메라 앱입니다.

운동, 식단, 공부, 출근, 여행처럼 “언제 어디서 무엇을 했는지” 남겨야 하는 순간에 바로 사용할 수 있습니다. 사진을 찍은 뒤 이미지에 어울리는 스탬프 스타일과 색상을 고르고, 토퍼처럼 꾸민 메모 템플릿을 더해 나만의 인증샷을 만들 수 있습니다.

주요 기능
- 시간과 날짜 스탬프
- 도로명 주소 기반 장소 스탬프
- 12시간/24시간 시간 표기 선택
- 사진 촬영 후 스탬프 스타일과 색상 변경
- 운동, 식단, 공부, 일상용 메모 템플릿
- 테니스, 골프 인증 문구 템플릿
- 메모 폰트, 색상, 윤곽선 설정
- 촬영한 사진 라이브러리 저장
- 폴더별 정리, 카테고리/시간대 필터
- 사진 저장 후 바로 공유

이런 분께 추천합니다
- 운동 인증샷을 꾸준히 남기고 싶은 분
- 식단, 공부, 출근 기록을 사진으로 관리하는 분
- 시간과 장소가 함께 보이는 사진이 필요한 분
- 단순한 타임스탬프보다 예쁜 스탬프 스타일을 원하는 분
- 테니스 코트, 골프 필드 상태를 기록하고 공유하는 분

사진 한 장에 시간, 장소, 메모를 깔끔하게 담아보세요.

## 키워드
인증샷, 인증샷 카메라, 타임스탬프, 시간 스탬프, 장소 스탬프, 위치 스탬프, 도로명 주소, 운동 인증, 식단 인증, 공부 인증, 골프 인증, 테니스 인증, 메모 카메라, 스탬프 카메라, 사진 기록

## 스크린샷 구성 제안
1. 촬영 화면: 시간/장소 스탬프가 실시간으로 보이는 화면
2. 촬영 후 편집: 무드별 스탬프 컬렉션과 색상 선택 패널
3. 메모 꾸미기: 템플릿, 폰트, 색상, 윤곽선 설정
4. 운동 템플릿: 테니스/골프 인증 문구 예시
5. 라이브러리: 폴더, 카테고리, 시간대 필터
6. 공유: 저장 후 바로 공유하는 흐름

## 스크린샷 카피 후보
- 시간과 장소가 자동으로 남아요
- 찍은 뒤 스탬프 스타일을 고르세요
- 메모는 토퍼처럼 예쁘게
- 운동 인증 문구도 바로 선택
- 라이브러리에서 폴더별로 정리
- 저장하고 바로 공유

## 개인정보/권한 설명 초안
카메라 권한은 사진 촬영을 위해 사용합니다.
위치 권한은 사용자가 장소 스탬프를 켠 경우 현재 위치 기반 주소를 표시하기 위해 사용합니다.
사진/미디어 권한은 촬영한 이미지를 저장하고 라이브러리에서 불러오기 위해 사용합니다.
